## Cross-Phase Integration (from gsd-integration-checker)

All wiring is **COMPLETE**. Every in-scope integration path is traced and confirmed:

- **Content flow:** `homepage.json` → index.astro → section components; `footer.json` → BaseLayout → HoursOverlay + Footer; `navigation.json` → NavBar + HamburgerOverlay; `ui-strings.json` → string lookups
- **Dual-mode:** `html[data-mode]` seeded by BaseLayout, mutated by 4 writers (StickyCTA, HamburgerOverlay, FooterColumns, ModeToggle), reflected by semantic tokens and component-level onsite overrides
- **Overlay triangle:** NavBar ↔ SearchOverlay ↔ HamburgerOverlay ↔ HoursOverlay cross-swap via `window.__{x}Overlay` globals, Lenis pause/resume wired on every open/close
- **Scroll stacking:** Lenis → ScrollTrigger.update → SectionWrapper pin + scale + hero-backdrop transition
- **Footer composition:** Marquee → Columns → Partners (new) → BottomBar, all fed from `footer.json`

No orphaned exports. No missing consumers. No broken flows.
