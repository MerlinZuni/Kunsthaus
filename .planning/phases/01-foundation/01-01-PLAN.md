<threat_model>
ASVS Level 1 assessment for a static site prototype:
- No user authentication, no server-side logic, no database -- minimal attack surface
- Font files self-hosted (no third-party font requests in production) -- no data leakage
- No cookies, no session management
- Leva dev overlay gated behind import.meta.env.DEV -- not shipped to production
- Content is static JSON rendered at build time -- no injection risk
- Threat: XSS via content injection -- mitigated by Astro's built-in HTML escaping in templates
- Threat: Dependency supply chain -- mitigated by pinning versions in package.json
No high-severity threats identified for this plan.
</threat_model>

         gsap.ticker.add((time) => { lenis.raf(time * 1000); });

         const motionQuery = window.matchMedia('(prefers-reduced-motion: reduce)');
         if (motionQuery.matches) { lenis.destroy(); }
         motionQuery.addEventListener('change', (e) => { if (e.matches) lenis.destroy(); });
