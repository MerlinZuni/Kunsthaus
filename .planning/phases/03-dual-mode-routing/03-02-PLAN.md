## Inherited Constraints (non-negotiable for every task)

1. **Base path `/Kunsthaus`.** Use `const base = import.meta.env.BASE_URL.replace(/\/$/, '');` then `${base}/onsite/...` or `${base}/images/...`. Pattern reference: `src/components/sections/ExhibitionCard.astro:15`, `src/components/sections/CinematicReveal.astro:20`.
2. **Dual-listener init for stateful scripts.** Every client `<script>` that binds handlers or runs layout measurement must call its `init()` from BOTH `document.addEventListener('DOMContentLoaded', init)` and `document.addEventListener('astro:page-load', init)`, and guard with `if (el.dataset.initialized === 'true') return; el.dataset.initialized = 'true';` to prevent double-binding on the first load.
3. **Dark palette via semantic tokens only.** `var(--color-surface)`, `var(--color-text)`, `var(--color-bg)`, or `color-mix(in srgb, var(--color-text) N%, transparent)`. No hex, no `rgb()`, no `white`/`black` keywords. The one documented exception is OnsiteHero grid lines (`rgba(255,255,255,0.08)`) per 03-CONTEXT.
4. **Reduce-motion guard on any GSAP.** `const reduceMotion = window.matchMedia('(prefers-reduced-motion: reduce)').matches || document.documentElement.classList.contains('reduce-motion');` — if true, set panel height instantly via `panel.style.height = 'auto' | '0'` without `gsap.to()`. Pattern reference: `src/components/sections/SectionWrapper.astro:54`.
5. **Event time format uses en-dash `–` (U+2013) not hyphen `-`.** All parsing must split on `'–'`. Grep `exhibitions.json` time fields to confirm.
6. **Two sibling SectionWrappers for the two building grids.** Never nest SectionWrappers.
7. **Do NOT modify `src/components/ui/StatusAccordion.astro`.** Its `DOMContentLoaded`-only gap is a latent 03-01 issue and is explicitly out of scope for 03-02.
8. **Do NOT modify `03-01-PLAN.md` or `03-01-SUMMARY.md`.**

**Script logic** (TypeScript inside `<script>`):
1. Define `init()` that queries all `[data-todays-events]` elements.

3. Parse events from `el.dataset.events`.
4. Compute `todayISO = new Date().toISOString().split('T')[0]`.
5. Filter events where `e.date === todayISO`. Sort by `time.split('–')[0]` (en-dash) minutes-since-midnight ascending.
6. Compute current minutes-since-midnight `nowMin = d.getHours()*60 + d.getMinutes()`.
7. For each event compute `startMin` and `endMin` by parsing `time.split('–')[0]` and `time.split('–')[1]` (watch the en-dash, NOT hyphen).
8. Determine state:
   - If `todays.length === 0` → state `none`, text = `lang === 'de' ? 'Heute keine Veranstaltungen' : 'No events today'`, dot class = `dim`.
   - Else find a "current" event where `startMin <= nowMin < endMin`. If found → state `now`, text = `lang === 'de' ? 'Jetzt: {title}' : 'Now: {title}'`, dot class = `live`.
   - Else if any event has `startMin > nowMin` → state `next`, find earliest, text = `lang === 'de' ? 'Als nächstes um {HH:MM} — {title}' : 'Next at {HH:MM} — {title}'`, dot class = `live`.

9. Set `.todays-events__text` innerText to computed string.
10. Populate `.todays-events__list` with one `<li>` per today event: `{time} — {title} · {type}` using `lang`.
11. Wire the trigger button: `addEventListener('click', ...)` toggles `aria-expanded`, toggles `aria-hidden` on the panel, and animates height open/closed. ## Anti-Patterns to Avoid

1. **Do not hardcode colors.** Every color through `var(--color-*)` or `color-mix()` on them. The one sanctioned exception is OnsiteHero's `rgba(255, 255, 255, 0.08)` grid lines.
2. **Do not use `DOMContentLoaded` alone in the accordion forks.** Bind both `DOMContentLoaded` and `astro:page-load`, guard with `dataset.initialized`.
3. **Do not hardcode `/onsite/...` or `/images/...` paths.** Always prefix with `${import.meta.env.BASE_URL.replace(/\/$/, '')}`.
4. **Do not animate or GSAP-pin OnsiteHero.** Static by spec.
5. **Do not fork `HeroCarousel.astro`.** OnsiteHero is ~30 lines.
6. **Do not reuse `ExhibitionList.astro`.** Wrong shape (list with hover-thumbnail).
7. **Do not build a real museum navigator.** Teaser only.
8. **Do not filter events at build time.** Client-side for "live" demo feel on any date.
9. **Do not forget reduce-motion gate on GSAP in accordions.** `window.matchMedia('(prefers-reduced-motion: reduce)').matches` → set height instantly, skip `gsap.to()`.
10. **Do not make the KJM card an `<a>` in 03-02.** 03-03 flips it when the target exists.
11. **Do not use hyphen `-` to split event time strings.** Events use en-dash `–` (U+2013).
12. **Do not nest SectionWrappers.** Two sibling wrappers for the two building groups.
13. **Do not modify `StatusAccordion.astro`, `03-01-PLAN.md`, or `03-01-SUMMARY.md`.** Scope discipline.
14. **Do not close sibling FAQ panels on open.** Multi-open required.
15. **Do not forget `target="_blank" rel="noopener noreferrer"` on the Museum Navigator external link.**
