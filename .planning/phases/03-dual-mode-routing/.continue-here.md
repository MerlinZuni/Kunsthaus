<decisions_made>

Key architectural/UX decisions this session:

- **ArtworkSlider crossfade stack over horizontal scroll-snap**. User wanted counter/arrows to visually stationary during slide changes. With identical per-slide counter/arrow rows stacked in one CSS grid cell (grid-area: stack), the crossfade makes them appear constant — only the number digit differs per slide. Simpler than a singleton overlay + sync JS.

- **Drag/swipe as pure input gesture, not scroll manipulation**. Native scroll-snap was fighting desktop drag release (stuck between slides). With no scroll at all, pointer events with a 10% directional threshold advance the index cleanly on both mouse and touch.

- **Portal modal to document.body**. Modal z-index: 100 was being capped inside SectionWrapper's stacking context, rendering the close button under the navbar scrim. Moving to body on init escapes the parent context.

- **Mobile slide locked to `height: 100svh` with explicit `grid-template-rows: 45svh auto 1fr`**. Auto rows + max-height:45svh let image heights vary by natural aspect, drifting counter position. Explicit sizing makes it structurally stable across slides.

- **CinematicReveal GSAP tween targets in px, not calc() strings**. Browser-resolved `margin: 0 auto` (px) can't be interpolated to `marginLeft: calc(-50vw + 50%)` (string), causing a visible jump at tween start. Computing `fullBleedMarginLeft = -(innerWidth - parentWidth) / 2` at build time lets GSAP tween px → px smoothly.

- **Shop CTA uses .btn.hover-wipe-inverse, not .brushed-brass**. The CSS comments explicitly label hover-wipe-inverse as "Primary CTA — dark glass pill" and brushed-brass as just "Brushed brass button". User called out that the brass should be treated as secondary/metallic, not primary.

- **CLAUDE.md Icons convention**. Material Symbols Outlined via `<span class="icon">glyph</span>` is the project default for icons. Inline SVG only acceptable when the glyph doesn't exist in Material Symbols or when the FILL variation axis is needed and the static WOFF2 subset doesn't support it. Documented with precedent pointers.
</decisions_made>

## Critical Anti-Patterns (do NOT repeat these)

- **Hand-drawn SVG icons** → Use `<span class="icon">glyph</span>` from Material Symbols Outlined. Fabricated SVGs look visually out-of-place next to the real glyphs on the site. Enforcement: CLAUDE.md §Icons convention.
- **Using `--space-10` token** → it doesn't exist in `src/styles/tokens/primitives.css`. Valid values: `1, 2, 3, 4, 6, 8, 12, 16, 24, 32`. Two bugs this session were caused by invalid token fallthrough to `auto`/`0`.
- **GSAP tweens that mix computed px with calc() string targets** → GSAP cannot interpolate between resolved pixel values and `calc()` expression strings, causing visible snaps at tween start. Always compute target values in px at timeline build time.
- **Trusting grid `align-content` default behavior** with stretched slides in `grid-area: stack` → caused subtle counter drift on mobile. Set explicit `align-content: start` and explicit row heights when a slide's layout must be structurally stable across all slide variants.
- **Adding components inside SectionWrapper without considering stacking context** → SectionWrapper creates its own z-index context. Modals/overlays that need to escape it must be portal'd to document.body on init.
