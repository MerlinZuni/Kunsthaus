# DINNextW1G Font Files

This directory should contain the following .woff2 font files:

- DINNextW1G-Light.woff2 (weight 300)
- DINNextW1G-Regular.woff2 (weight 400)
- DINNextW1G-Medium.woff2 (weight 500)
- DINNextW1G-Bold.woff2 (weight 700)

DINNextW1G is a commercial font from Linotype. Font files must be obtained
from the project's Figma design or font license provider.

Until font files are added, the site will use the fallback font stack:
'DIN Alternate', 'DIN Next', system-ui, -apple-system, sans-serif
